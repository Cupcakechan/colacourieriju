# Cola Courier: Iju

A top-down arcade cola-delivery game for the **Ultimate AI-Powered Game Jam #1** (mandatory theme: **COLA**). Play as **Iju**, a yokai mountain courier, and keep the fizz settled while you deliver across a misty mountain town.

## Run locally

No build step. From this folder, in a terminal:

```
npx serve
```

Then open the printed URL (e.g. `http://localhost:3000`). The first run will ask to install `serve` — accept it.

> ES modules must be served over http — opening `index.html` directly as a `file://` won't work.

## Controls

- **Move:** WASD / Arrow keys
- **Jump:** Space
- **Restart:** R (on the score screen)
